# 使用TypeScript封装弹窗视频组件
## 功能
- 点击弹窗
- 可配置弹窗位置：居中|居左|居右
- 视频弹出时，遮罩出现
- 可以配置自动播放
- 进度条拖动
- 点击音量图标静音与恢复默认音量
- 鼠标移出组件，自动隐藏进度条

## 将来也许可以完善的功能
- 添加视频封面
- 点击进度条跳转
- 支持倍速播放
- 线路选择
- 美化组件
